# 文献结论（Schoenfeld et al., 2017，J Sports Sci，系统综述+元分析）
- 15 项研究、34 个训练组；以每肌群“周总组数”为自变量，肌肥大为因变量做元回归。
- 每增加 1 组（/周/肌群），效果量提升约 0.023（约 0.37% 增长）；同研究内高 vs 低组数差异 ES≈0.241（≈3.9% 增长）。
- 分档（<5、5–9、≥10 组）显示 10+ 组趋势最佳（P≈0.074）；证据覆盖至中高量，超高量证据有限。
- 对应训练哲学：TRAINING_PHILOSOPHY.md 4.1.4（剂量-反应曲线与可恢复区间）。

# 训练与疲劳管理要点
- 实操参考：≥10 组/周/肌群是中高量下限，想追求更快增肌可上探 12–20 组，但需分频（≥2–3 次/周）避免单次堆量。
- 避免单次堆量：一次做太多组，后半程质量滑坡，变“垃圾容量”；分散到多次训练+足够休息能维持张力与动作质量。
- 监控输出：若同重速度/次数掉 20–25% 或 RIR 提前耗尽，优先延长休息或减组；必要时安排降载周。
- 肌群差异：复合动作已高刺激的协同肌（如二头在拉练中）不必极端堆量；复合动作覆盖不足的肌群（如三头长头、侧束）可在中高量内微调。

# 神经压力/恢复说明
- 高量叠加短休会加重中枢/外周疲劳，技术易崩并拉长恢复，削弱后续训练质量。
- 以高质量有效组为目标：宁可在每组保持张力和动作质量，也不要用低质量的“填充组”堆数量。

# 本文 takeaway
- 元分析支持“多组更好”：≥10 组/周/肌群通常优于更低组数，每多 1 组有小幅收益。
- 证据覆盖到中高量，超高量未充分验证；想加量必须分频、保质量，防止后半程变垃圾容量。
- 质量优先：当输出下降或疲劳飙升时，延长休息/减组/降载，比盲目堆量更能促进有效刺激。
